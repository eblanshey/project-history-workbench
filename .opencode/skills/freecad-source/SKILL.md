---
name: freecad-source
description: Inspect the freecad source code
---

The freecad source repository can be found one directory up from this project. Just go up one directory from the root project, and cd into `freecad-source`, e.g. `cd ../freecad-source`. You may grep and search through files to see how it implements any functionality. You may instruct the speedy "explore" subagent to find relevant files, just make sure to give them the full path to the correct directory, as it's outside of this project.
